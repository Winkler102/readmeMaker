  ## Installation
  Prerequisite: Node.js.  
    
     1. Clone from repository  
     2. Open command prompt to folder  
     3. Run npm install.    
    
   ![Install Instruction Video](https://youtu.be/4VQ2QT3zbM8)

  
  ## Usage 
    
   1. Open command prompt to folder  
   2. Run node index.
   3. Answer all prompts.
